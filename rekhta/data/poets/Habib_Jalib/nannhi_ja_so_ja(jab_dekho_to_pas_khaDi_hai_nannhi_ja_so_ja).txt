jab dekho to paas khaḌī hai nannhī jā so jā
tujhe bulātī hai sapnoñ kī nagrī jā so jā

ġhusse se kyuuñ ghuur rahī hai maiñ aa jā.ūñgā
kah jo diyā hai tere liye ik guḌiyā lā.ūñgā
ga.ī na zid karne kī aadat terī jā so jā
nannhī jā so jā

in kaale darvāzoñ se mat lag ke dekh mujhe
uḌ jaatī hai niiñd āñkhoñ se pā kar paas tujhe
mujh ko bhī sone de merī pyārī jā so jā
nannhī jā so jā

kyuuñ apnoñ aur begānoñ ke shikve kartī hai
kyuuñ āñkhoñ meñ aañsū lā kar aa.heñ bhartī hai
rone se kab raat kaTī hai dukh kī jā so jā
nannhī jā so jā
